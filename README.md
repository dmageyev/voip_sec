# voip_sec

## Технології VoIP — презентація для магістрів

Файл **[presentation.html](presentation.html)** містить повну слайдову презентацію «Технології VoIP» для студентів магістратури. Це повністю автономний HTML-файл без зовнішніх залежностей — реалізований на чистому HTML/CSS/JavaScript.

### Як переглянути

Відкрийте `presentation.html` у будь-якому сучасному браузері (Chrome, Firefox, Edge, Safari).  
Інтернет-з'єднання **не потрібне**.

### Керування

| Клавіша / дія | Функція |
|---------------|---------|
| `→` / `↓` / `Пробіл` | Наступний слайд |
| `←` / `↑` | Попередній слайд |
| `F` | Повноекранний режим |
| Свайп (тач-екран) | Наступний / попередній слайд |
